Scalar derivatives

Some notation for scalar derivatives:

  • For a function f∈R→Rf\in\mathbb R\to \mathbb Rf∈R→R, we write f′∈R→Rf'\in\mathbb R\to Rf′∈R→R for its derivative with respect to its argument. If the argument is called xxx, we can also write ddxf\frac{d}{dx} fdxd​f. If the argument represents time, we sometimes write f˙\dot ff˙​.
  • If a function depends on more than one variable, we write ∂∂xf\frac{\partial}{\partial x} f∂x∂​f or ∂∂yf\frac{\partial}{\partial y} f∂y∂​f to indicate a partial derivative: the derivative with respect to one variable while holding the others constant.
  • Second and higher derivatives are f′′f''f′′, f¨\ddot ff¨​, d2dx2f\frac{d^2}{dx^2} fdx2d2​f, or ∂2∂x∂yf\frac{\partial^2}{\partial x\partial y} f∂x∂y∂2​f.
  • For a function fff, we write f∣x^f\big |_{\hat x}f∣∣​x^​ or f(x)∣x=x^f(x)\big |_{x=\hat x}f(x)∣∣​x=x^​ to represent evaluation at x^\hat xx^. This means the same thing as f(x^)f(\hat x)f(x^) but is sometimes clearer: it lets us keep one name (xxx) for the variable we are differentiating, and another name (x^\hat xx^) for the value we are substituting at the end.

Scalar identities

Some of the most common identities for working with scalar derivatives:

  • Differentiation and partial differentiation are linear operators: for example, (af+bg)′=af′+bg′(af+bg)' = af' + bg'(af+bg)′=af′+bg′.
  • Chain rule: if we want ddxf(g(x))\frac{d}{dx}f(g(x))dxd​f(g(x)), then we use
    dfdx=dfdgdgdx\frac{df}{dx} = \frac{df}{dg} \frac{dg}{dx}dxdf​=dgdf​dxdg​
    (As a mnemonic, we can "cancel the dgdgdg" — but since dfdg\frac{df}{dg}dgdf​ isn't really division, this is just a mnemonic.) Another way to write the same thing:

Vector derivatives

It's also useful to think about functions that return vectors or take vectors as arguments. If fff is a vector-valued function of a real argument, f∈R→Rnf\in\mathbb R\to\mathbb R^nf∈R→Rn, we can write it as a vector whose components are real-valued functions,

f(x)=(f1(x)f2(x)⋮fn(x))f(x) = \left(\begin{array}{c}f_1(x)\\ f_2(x)\\ \vdots \\ f_n(x) \end{array}\right)f(x)=⎝⎛​f1​(x)f2​(x)⋮fn​(x)​⎠⎞​

Its derivative is then also a vector-valued function, of the same shape as fff. Its components are the derivatives of the component functions:

ddxf=(df1dxdf2dx⋮dfndx)\frac{d}{dx} f = \left(\begin{array}{c}\frac{df_1}{dx}\\[1ex] \frac{df_2}{dx}\\ \vdots \\[.5ex] \frac{df_n}{dx} \end{array}\right)dxd​f=⎝⎛​dxdf1​​dxdf2​​⋮dxdfn​​​⎠⎞​

We can think of fff as representing a curve in Rn\mathbb R^nRn. The derivative dfdx\frac{df}{dx}dxdf​ represents a tangent vector to this curve: the instantaneous velocity of a point moving along the curve as the argument xxx changes at a unit rate. The length of the tangent vector tells us the speed of the point, and the components tell us its direction.

Chain rule for vectors

With the above notation, the chain rule for vector functions looks just like it did for scalar functions. Suppose f∈Rn→Rf\in\mathbb R^n\to\mathbb Rf∈Rn→R takes multiple arguments and g∈R→Rng\in\mathbb R\to\mathbb R^ng∈R→Rn returns multiple values, so that f(g(x))f(g(x))f(g(x)) makes sense. Then we have

dfdx=dfdgdgdx\frac{df}{dx} = \frac{df}{dg} \frac{dg}{dx}dxdf​=dgdf​dxdg​

This looks just like the scalar chain rule (we "cancel the dgdgdg"). But now dfdg\frac{df}{dg}dgdf​ is a row vector in R1×n\mathbb R^{1\times n}R1×n and dgdx\frac{dg}{dx}dxdg​ is a column vector in Rn×1\mathbb R^{n\times 1}Rn×1, so that when we multiply them we get their dot product. For clarity we can indicate the values of the arguments to each function:

dfdx∣x=dfdg∣g(x) dgdx∣x\frac{df}{dx}\bigg |_x = \frac{df}{dg}\bigg|_{g(x)}\, \frac{dg}{dx}\bigg|_xdxdf​∣∣​x​=dgdf​∣∣​g(x)​dxdg​∣∣​x​

If we write out the dot product, we get

dfdx=∑i=1n∂f∂gidgidx\frac{df}{dx} = \sum_{i=1}^n \frac{\partial f}{\partial g_i} \frac{dg_i}{dx}dxdf​=i=1∑n​∂gi​∂f​dxdgi​​

which may be familiar as the rule for calculating the total derivative of fff with respect to xxx. In words, to calculate the change in fff, we sum up the effects of all of the changes in all of the inputs to fff.
